# 🛒 Market Basket Analysis

## 📌 Project Overview
This project performs Market Basket Analysis using association rule mining to identify relationships between products.

## 📂 Dataset
- File: store_data.csv
- Contains transaction data of customers

## 🛠 Tools & Technologies
- Python
- Pandas
- Mlxtend
- Jupyter Notebook

## 📊 Method Used
- Apriori Algorithm
- Association Rules

## 📈 Output
- Frequent itemsets
- Strong product associations
